# TalentWrap - Resume Profile Generator

This application generates a professional profile summary by merging user-provided details with an uploaded resume.

## Deployment on Hugging Face Spaces

1. Create a new Space on Hugging Face.
2. Select **Docker** as the SDK.
3. Choose the **Blank** template or connect your GitHub repository.
4. Ensure the `Dockerfile` exposes port `7860`.
